Rockibot

Get Started

Built by Rocky.

Rockibot is a Discord bot created using Discordjs/Commando and Discord.js. It is a extremely modular bot for those who want some features, and only one bot for it.

Versatile

Rockibot is extremely versatile, being extremely modular. Each server can have mutiple modules, and each server will be able to have a different layout for their bot!

Basic Commands

These commands are global for all servers. This inclues the commands !ping, !invite, !stats and !status.

Moderation

These commands are from the first module for the bot. This inclues the commands !ban, !kick, !modlog, !mute, !purge, !unmute, !warn, !unwarn and !warns.

Suggesions

These commands are for suggestions for servers that have enabled the module. This inclues the commands !suggest-channel, !suggestion and !approve/consider/implement/deny.

Music

These commands are for music commands for servers that have enabled the module. This inclues the commands !join, !leave, !play and !pause.
